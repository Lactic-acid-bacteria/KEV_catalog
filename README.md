## 使い方
お好きなディレクトリにcd してスクリプト実行してください。<br>
.\KEVに実行時のKEVがDLされ、すでにある「before_KEV.json」と比較し、差分抽出されます。※ファイル名がとてもややこしくてごめんなさい。。。<br>
差分に関するファイルが出力されるので、加工してください。<br>
## 例
私はNVDでCVEidを検索して社内SNSに投げています。
